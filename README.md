SteamVault:

SteamVault is a game library manager built with Django. It allows users to keep track of their game collection by adding, editing, viewing, and deleting games through a simple and clean interface.

Features

- Add new games to your library
- Edit existing game information
- Delete games from the library
- View detailed information about each game
- Dynamic homepage with real-time statistics
- Categorize games by status (Playing, Completed, Wishlist)
- Responsive and modern UI

Tech Stack

- Python
- Django
- SQLite
- HTML
- CSS
